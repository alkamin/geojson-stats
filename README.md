# Usage
<!-- usage -->
```sh-session
$ npm install -g geojson-stats
$ geojson-stats COMMAND
running command...
$ geojson-stats (-v|--version|version)
geojson-stats/0.0.0 darwin-x64 node-v13.6.0
$ geojson-stats --help [COMMAND]
USAGE
  $ geojson-stats COMMAND
...
```
